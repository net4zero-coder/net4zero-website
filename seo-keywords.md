## Filozofia doboru

Frazy są dobierane pod **3 intencje wyszukiwania** decydenta:
1. **Informacyjne** ("co to system kaucyjny", "ustawa kaucja 2025") → wciąga ruch top-of-funnel, edukuje
2. **Porównawcze** ("recyklomat dla gminy", "system kaucyjny koszt") → środek lejka, person decyzyjnych
3. **Transakcyjne** ("oferta recyklomat miasto", "wdrożenie systemu kaucyjnego przetarg") → bottom-of-funnel, gorące leady

Dla każdej frazy wskazany **landing page** i **typ contentu** (LP = landing page, B = blog, F = FAQ, P = strona produktu).

## Plan implementacji SEO (techniczny)

**Etap 1 (start projektu):**
- Title + meta description unikalne dla każdej z 7 stron
- Schema.org: `Organization`, `LocalBusiness`, `Service` (Pakiet Miasto), `FAQPage` (FAQ przetargowe), `Article` (blog)
- Open Graph + Twitter Cards
- Sitemap.xml + robots.txt
- Google Search Console + GA4 + Microsoft Clarity od dnia 1

**Etap 2 (1-3 m-c po starcie):**
- 1 artykuł blogowy / mc pod long-tail (lista C)
- Optymalizacja zdjęć (WebP, lazy load, alt-texty)
- Link building: cytowanie w artykułach branżowych, gościnne wpisy

**Etap 3 (3-6 m-c):**
- Pomiar pozycji + iteracja
- Lokalne podstrony jeśli pojawią się realne lokalizacje
- Backlinki z BDO, ministerialnych portali
